# 共通編集基準

本サイトの全バーティカルに共通する編集基準。
inquire.jp エディトリアルシステムを基盤として策定。

## 文体

| 項目 | 規定 |
|------|------|
| 文末 | だである調（常体） |
| 人称 | 三人称基本 |
| 語り口 | 落ち着いた、知的好奇心のある |
| テンション | 中庸。過度な興奮も冷淡さも避ける |

## 表記ルール

| 項目 | ルール |
|------|--------|
| AI表記 | 「AI」に統一 |
| 英数字 | 半角 |
| カッコ | 「」会話・引用、『』書名・作品名 |
| 三点リーダー | 「……」二つ連続 |
| ダッシュ | 「——」二つ連続 |
| 読点 | 「、」使用 |

| 敬称 | 記事本文は「さん」付け。初出はフルネーム＋肩書き |

## メタファー

### 使ってよいメタファー（ガーデニング/生態系系）
- 種をまく、芽が出る、根を張る、花が咲く、実を結ぶ
- 土壌を耕す、栄養を与える、生態系を育む
- 受粉する、循環する、共生する
- 季節の移ろい、成長と休眠のサイクル

### 禁止メタファー
- 戦争系: 戦略、撃破、征服、武器、弾丸、戦場
- 機械系: 歯車、エンジン、燃料、加速装置
- 競争系: 勝敗、レース、追い抜く、圧倒する

## 記事長さの基準

| ティア | 文字数 | 用途 |
|--------|--------|------|
| brief | 500〜800字 | 速報的な短いニュース |
| standard | 1,000〜1,500字 | 通常のニュース記事 |
| deep | 1,500〜2,500字 | 深掘り・分析記事 |

## レビューチェックリスト

### 事実性（Accuracy）
- 固有名詞（人名・社名・地名）は正確か
- 数値・日付は確認済みか
- 引用は原文と照合済みか

### コンセプト整合性（Concept Alignment）
- バーティカルの editorial.angle に沿っているか
- forbidden_framing に違反していないか

### ブランドボイス（Brand Voice）
- 誠実・好奇心・敬意・温かさ・地に足——の特性に沿っているか
- 禁止メタファーを使用していないか
- 過度な美化、誇張、センセーショナルな表現がないか

### 表記・文体（Style）
- だである調で統一されているか
- 表記ルールに準拠しているか

### 構成・読みやすさ（Structure & Readability）
- テンプレートの構成に沿っているか
- 段落が長すぎないか（1段落200〜300字目安）
- 文字数が規定範囲内か

### 読者価値（Reader Value）
- 読了後に得られるもの（知見・視座・インスピレーション）が明確か
- 行動につながるヒントや問いが含まれているか

## イベント紹介記事の追加ルール

ニュース記事と共通の基準に加え、イベント紹介記事には以下の追加ルールを適用する。

### 時制

- **未来形を基調とする**: 開催前のため「開催される」「議論される予定である」等の表現を使う
- 確定情報（日時・会場・登壇者）と見立て（注目ポイント・期待される議論）を明確に区別する

### 禁止表現

- 過度な参加促進: 「必見」「絶対に参加すべき」「見逃せない」等
- 未開催の内容を事実として断定: 「〜が議論された」「〜が明らかになった」等

### 記事構成

| セクション | 文字数目安 | 内容 |
|-----------|-----------|------|
| イベントの概要 | 300〜500字 | 目的・背景・主催者・プログラム |
| なぜ今、このテーマか | 300〜500字 | 社会的背景・ベタービジネス接続 |
| 注目のポイント | 300〜500字 | 見どころ・登壇者の専門性 |
| 開催情報 | — | 構造化データ（リスト形式） |

### 開催情報の記載フォーマット

```markdown
- **イベント名**: 正式名称
- **日時**: YYYY年MM月DD日（曜日）HH:MM〜HH:MM
- **会場**: 会場名（住所） or オンライン（使用ツール）
- **参加費**: 無料 or 金額
- **定員**: XX名（わかれば）
- **申込**: [申込ページ](URL)
```

### 追加チェック項目（イベント固有）

- [ ] 開催日が未来の日付であるか
- [ ] 申込URLが有効で正しいページに遷移するか
- [ ] 未開催のイベント内容を事実として断定していないか
- [ ] 過度な参加促進表現がないか
